# 子供向けの足し算練習Web

小学一年生向けの簡単な足し算・引き算の練習Webアプリです。
一問ずつ表示して答えていき20問くらいで正解数を表示します。
子供の興味をひくように楽しいエフェクトが出ます。

## 単純足し算モード

ひとけたの足し算を20問表示します。

## 単純引き算モード

ひとけたの引き算を20問表示します。
答えは常にプラスになるようにします。

## 文章題モード

生成AIを使って一桁の足し算・引き算になるような文章題を考えてください。
小学生が興味を持つような内容にしてください。
文は以下の例を参考に生成AIで作ってください。

例：

- いちごが 3 こ あります。そこへ、あたらしく いちごが 2 こ ふえました。いちごは ぜんぶで なんこ に なりましたか？
- みかんが 7 こ あります。そのうち 3 こ たべました。みかんは なんこ のこって いますか？

## 技術スタック

Cloudflareを活用しインフラコストを最小になるようにしてください。
DBは使わずにLocal Storageに情報を保存するようにしてください。
スマホ・タブレットで使うことを想定し、答えをワンタップでボタンで入力できるようにしてください。
